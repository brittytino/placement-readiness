## Overview

This repository contains my submission for Mission Tata Day 1: Git & Collaboration.

The objective of this activity was to understand and successfully perform the complete Git workflow used in real-world software development environments.

## Git Workflow Completed

The following operations were performed:

1. Forked the main repository to my GitHub account.
2. Cloned the forked repository to my local machine.
3. Created my personal folder using my roll number.
4. Added my profile information using `profile.md`.
5. Committed the changes with a meaningful commit message.
6. Pushed the changes to my forked repository.
